After creating the repository on GitHub, run these commands:

git remote add origin https://github.com/Hlomohangcue/communication-bridge-ai.git
git push -u origin master

When prompted for credentials:
Username: Hlomohangcue
Password: Use a Personal Access Token (create at https://github.com/settings/tokens)

To create a Personal Access Token:
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Name: "Communication Bridge AI"
4. Check "repo" scope
5. Click "Generate token"
6. Copy the token and use it as your password
